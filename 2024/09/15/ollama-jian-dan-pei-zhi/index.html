Ollama简单配置

Linux LLM
教程
Publish Date:   2024-09-15

Ollama介绍

Ollama是一个创建、运行和管理大语言模型(LLM)的工具。

Ollama安装

  1. 官网：ollama
  2. 下载安装资源包并运行安装程序
  3. 输入ollama验证安装

    Ollama启动

  4. 一般启动Ollama：ollama serve
  5. Linux设置Ollama为自启服务：StartupService

2.1. 创建Ollama用户与用户组

sudo useradd -r -s /bin/false -U -m -d /usr/share/ollama ollama
sudo usermod -a -G ollama $(whoami)

2.2. 创建服务配置文件于/etc/systemd/system/ollama.service

[Unit]
Description=Ollama Service
After=network-online.target
[Service]
ExecStart=/usr/bin/ollama serve
User=ollama
Group=ollama
Restart=always
RestartSec=3
Environment="PATH=$PATH"
[Install]
WantedBy=default.target

2.3. 启动服务

sudo systemctl daemon-reload
sudo systemctl enable ollama
sudo systemctl start ollama
sudo systemctl status ollama

下载训练好的大模型

  1. 在资源网站找到对应大模型：Models
  2. 运行介绍页面命令下载大模型ollama run ***

    设置局域网访问(Linux)

  3. 设置Ollama为自启服务启动
  4. 在/etc/systemd/system/ollama.service文件中Service条目添加如下代码
    Environment="OLLAMA_HOST=0.0.0.0:11434"
Environment="OLLAMA_ORIGINS=*"
  5. 重启服务
    systemctl daemon-reload
systemctl restart ollama
    其他平台参考：Setting environment variables

    卸载(Linux)

  6. 如果设置了Ollama服务
    sudo systemctl stop ollama
sudo systemctl disable ollama
sudo rm /etc/systemd/system/ollama.service
sudo userdel ollama
sudo groupdel ollama
  7. 删除Ollama二进制文件
    sudo rm $(which ollama)
  8. 删除下载的大模型
    sudo rm -r /usr/share/ollama
