端口理解

1.计算机之间的通信实质上是不同计算机之间进程的通信，而端口号就是唯一的表示这些进程的编号，至于进程，可以理解为运行着的程序。

2.端口号有16位，0到65535，一共有2^16个端口号。通过端口号可以找到计算机运行的应用程序，联系网络安全，端口攻击。

3.在Internet上，按照协议类型分类，端口被分为TCP端口和UDP端口两类，虽然他们都用正整数标识，但这并不会引起歧义，比如TCP的80端口和UDP的80端口，因为数据报在标明端口的同时，还将标明端口的类型

4.从端口的分配来看，端口被分为固定端口和动态端口两大类（一些教程还将极少被用到的高端口划分为第三类：私有端口。像这样分类：0-1024公共端口，1025-49151注册端口，49152-65535动态和私有端口）：
4.1固定端口（0－1023）：使用集中式管理机制，即服从一个管理机构(IANA)对端口的指派，这个机构负责发布这些指派。（低于255用于公共应用，255-1023分配给各个公司来用于特殊应用）由于这些端口紧绑于一些服务，所以我们会经常扫描这些端口来判断对方 是否开启了这些服务，如TCP的20/21(ftp)、80(http)、139(netbios)、23(telnet)、25(smtp)、53(dns)，UDP的7(echo)、69(tftp)、161(snmp)
4.2动态端口（1024－49151）：这些端口并不被固定的捆绑于某一服务，操作系统将这些端口动态的分配给各个进程， 同一进程两次分配有可能分配到不同的端口。不过一些应用程序并不愿意使用操作系统分配的动态端口，他们有其自己的‘商标性’端口，如oicq客户端的 4000端口，木马冰河的7626端口等都是固定而出名的。

2.UIControl客户端总控：调用界面类控制界面，UDP监听端口即为Python程序占用的端口

from UI.ChatUI import ChatUI
from UI.LoginUI import LoginUI
import tkinter as tk
import socket
import threading
 
class UIControl:
 
    def __init__(self):
        self.key=False
        self.t=threading.Thread(target=self.checkMessage)
        self.t.daemon = True
        self.s=socket.socket(socket.AF_INET,socket.SOCK_DGRAM)
        self.host_port=5005
        self.host='127.0.0.1'
        self.address='127.0.0.1'
        self.port=1111
        self.s.bind((self.address,self.port))
        self.c=ChatUI(self.sendMsg)
        self.l=LoginUI(goChat=self.login)
        self.l.show()
 
    def __del__(self):
        try:
            if self.key:
                msg=self.address+"<"+str(self.port)+"<"+"quit"
                self.sendMessage(msg)
            self.s.close()
        except Exception as e:
            print(e)

    def sendMsg(self):
        msg=self.id+"<"+self.c.msg.get()
        self.sendMessage(msg)

    def sendMessage(self,msg):
        self.s.sendto(msg.encode("utf-8"),(self.host,self.host_port))

    def recvMessage(self):
        return self.s.recvfrom(1024)

    def login(self):
        msg=self.address+"<"+str(self.port)+"<"+self.l.id.get()+"<"+self.l.password.get()
        self.sendMessage(msg)
        data,addre=self.recvMessage()
        c=data.decode().split('<')
        if c[0]=="password" and c[1]=="ok":
            self.id=self.l.id.get()
            self.t.start()
            self.l.hide()
            self.c.show()
            self.key=True
        else:
            self.l.cue.set("Someting is wrong")
 
    def checkMessage(self):
        try:
            while True:
                data,addre=self.recvMessage()
                c=data.decode().split('<')
                self.c.text.insert('end',c[0]+"说："+c[1]+"\n")
        except Exception as e:
            print(e)
if __name__ == '__main__':

3.LoginUI登录界面类：输入服务器存储的账户信息来登录客户端（建立连接）

import tkinter as tk


class LoginUI:
    def __init__(self, goChat):
        self.goMain = goChat
        self.showKey = False

    def show(self):
        self.showKey = True
        self.window = tk.Tk()
        self.id=tk.StringVar()
        self.password=tk.StringVar()
        self.cue=tk.StringVar()
        self.window.title("LoginUI")
        self.window.geometry('250x150')
        tk.Label(self.window,text="账号：").grid(row=1,column=1)
        tk.Entry(self.window,textvariable=self.id).grid(row=1,column=2)
        tk.Label(self.window,text="密码：").grid(row=2,column=1)
        tk.Entry(self.window,textvariable=self.password).grid(row=2,column=2)
        tk.Button(self.window, text="登录",
                  command=self.goMain).grid(row=3, column=2)
        tk.Label(self.window,textvariable=self.cue).grid(row=4,column=1)
        self.window.mainloop()

    def hide(self):
        if self.showKey:
            self.window.destroy()
            self.showKey = False

4.ChatUI聊天界面：实时接收信息，可以编辑信息发送，涉及开辟线程

import tkinter as tk


class ChatUI:
    def __init__(self,send):
        self.showKey=False
        self.send=send

    def show(self):
        self.showKey=True
        self.window = tk.Tk()
        self.msg=tk.StringVar()
        self.window.title("ChatUI")
        self.window.geometry('450x250')
        self.text=tk.Text(self.window)
        self.text.grid(row=1,column=1)
        tk.Entry(self.window,textvariable=self.msg).grid(row=2,column=1)
        tk.Button(self.window, text="发送",
                  command=self.send).grid(row=2, column=2)
        self.window.mainloop()
    def hide(self):
        if self.showKey:
            self.window.destroy()
            self.showKey=False
